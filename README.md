# District Planning & Monitoring System (DPMS)

Version : 1.0

A web application for District Panchayat to monitor weekly planning and achievements of all district departments.

## Technology

- HTML5
- Bootstrap 5
- Vanilla JavaScript
- Supabase
- PostgreSQL
- Chart.js

## Modules

- Login
- Dashboard
- Weekly Action Plan
- Action Items
- Progress Updates
- Reports
- Department Management
- User Management
- Settings

## User Roles

- DDO
- HOD
- Viewer

## Developed For

District Panchayat Administration
